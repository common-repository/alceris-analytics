## Setup this WordPress plugin

== Installation ==

1. Log into Wordpress
2. Go to `Plugins`, `Add New`
3. Search for `Alceris Analytics`
4. Click `Install Now`, then `Activate`
5. Go to `Settings`, then `Alceris Analytics`
6. Enter your Site ID. It can be found in the "Website integration" tab on the [Alceris dashboard](https://alceris.com/dashboard)
